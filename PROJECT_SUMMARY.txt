PROJECT OVERVIEW
================================================================================

A full-stack web application that:
  ✓ Randomly pairs users together for two-player games
  ✓ Establishes peer-to-peer voice chat via WebRTC
  ✓ Allows players to select from 6 available games
  ✓ Provides real-time audio indicators
  ✓ Features a modern, responsive UI

QUICK START (3 STEPS)
================================================================================

1. npm install
2. npm start
3. Open http://localhost:3000 (in multiple tabs to test)

HOW IT WORKS
================================================================================

1. User joins lobby with username
2. User enters queue for match
3. When 2+ users in queue, they're auto-paired
4. Both users see available games
5. Users select a game (shared in real-time)
6. Game starts → WebRTC voice connection initiates
7. Audio indicators show who's speaking
8. Users can toggle microphone/speaker

COMMON ISSUES
================================================================================

Microphone not working?
  → Check browser permissions (F12 Settings)
  → Check Windows/Mac microphone settings
  → Ensure microphone is plugged in

Users not pairing?
  → Verify both in queue
  → Check browser console (F12)
  → Refresh and try again

Voice won't connect?
  → Wait 5-10 seconds for WebRTC
  → Check console for errors
  → Both need microphone permission

Port already in use?
  → PORT=3001 npm start
